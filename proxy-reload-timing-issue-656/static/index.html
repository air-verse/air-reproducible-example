How to Reproduce the Issue:

  1. Make sure Air is running with air command
  2. Access this page through the proxy at http://localhost:8081
  3. Edit main.go (add a comment or change something)
  4. Save the file to trigger Air's hot reload
  5. Observe: Browser will try to reload immediately, but server takes 2+ seconds to start
  6. Result: You'll see "Proxy handler: unable to reach app" error
  7. Manual refresh is required after server is ready
